# Sandbox Python Projects

This folder contains small Python practice projects. Each script is standalone and can be run from the terminal with Python 3.

## Projects

| File | Description |
| --- | --- |
| `caesar_cipher.py` | Encrypts or decrypts messages using a Caesar cipher shift. Preserves punctuation, spaces, numbers, and non-ASCII characters. |
| `hangman.py` | A command-line Hangman game with random words, ASCII art, repeated-guess handling, and win/loss states. |
| `calculator.py` | A command-line calculator that supports addition, subtraction, multiplication, division, continued calculations, fresh starts, and exit. |
| `secret_auction.py` | A secret auction program that collects bidder names and bid amounts, then announces the highest bidder. |
